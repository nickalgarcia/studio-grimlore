# **App Name**: Grimlore Forge

## Core Features:

- Decision Input: Allow the DM to input the party's actions and decisions as context for idea generation.
- Context-Sensitive Idea Generation: Generate plot hooks, encounter ideas, and NPC concepts based on the party's recent actions, powered by a generative AI tool.
- Concept Library: Enable DMs to save generated ideas to a personal library for later use and reference.
- Inspiration Prompt Generator: Generate random prompts to spark the DM's creativity and break through writer's block.

## Style Guidelines:

- Primary color: Deep purple (#4B0082) to evoke a dark fantasy atmosphere.
- Background color: Forest green (#228B22) desaturated to 20% to provide a dark, natural backdrop: #1A611A.
- Accent color: Metallic gold (#FFD700) used sparingly for highlights and interactive elements.
- Body text and headline font: 'Literata' for a literary, vintage, slightly formal feel.
- Use icons with a hand-drawn, aged appearance to enhance the dark fantasy aesthetic.
- Design the layout with a parchment-like background texture and ornate borders to evoke a sense of ancient tomes.
- Subtle animations for transitions and reveals to maintain user engagement.
